# TurtleBot Documentation

This repository is intended to run a [**TurtleBot 4**](https://clearpathrobotics.com/turtlebot-4/) in the [Signal Processing and Robotics Lab at the University of Applied Sciences Esslingen Germany](https://www.hs-esslingen.de/forschung/forschungsaktivitaeten/forschungsbereiche/iis-institut-fuer-intelligente-systeme).

In oder to get a deep insight into this project and to use the TurtleBot4 for your custom tasks, follow the steps below.

1. Check out the [Quickstart Guide](quickstart.md) to set up the development environment. Two options are provided:

    * [VNC and Visual Studio Code devcontainers](howToVNC.md) (**this is the recommended option for the ASD lab**), and 
    * [xpra](howToXpra.md).

2. Once everything has been set up successfully, try the [TurtleBot simulation](simulation.md) in Gazebo.

3. Finally, test the [real TurtleBot 4](real_turtlebot.md).

4. Notes on using a GPU can be found in the [devcontainer README](../../.devcontainer/README.md).


### Some Important Resources

* The [TurtleBot 4 user manual](https://turtlebot.github.io/turtlebot4-user-manual/) with tutorials on driving, simulation, and reading sensors.
* The [ROS2 Jazzy Documentation](https://docs.ros.org/en/jazzy/index.html).
